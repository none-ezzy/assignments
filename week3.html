Issues Encountered

During my playthrough of the Command Line Dungeon Adventure, I faced several challenges:

  • Navigation Confusion: Initially, I struggled with the correct command syntax to navigate between directories, especially when moving up multiple levels.
  • Key Management: Finding and using the StoneKey.key file to unlock the LockedDoor was not intuitive at first.
  • Hidden Commands: Some puzzle solutions required using commands that weren't explicitly listed in the instructions.
  • Path Tracking: Keeping track of my location within the complex directory structure was challenging without a visual aid.

Research Conducted

To overcome these challenges, I had to research:

  • Advanced command line navigation techniques using relative and absolute paths
  • How to use the attrib command to modify file attributes
  • Methods for searching text within files using findstr
  • Techniques for copying and moving files between directories

I discovered that the cd / command could quickly return me to the root directory, which proved invaluable when I got lost in the dungeon structure.

Where I Found Help

I utilized the following resources to complete the assignment:

  • Microsoft's Command Line Reference - For detailed command syntax
  • SS64 Command Line Reference - For quick command lookups
  • Course textbook chapters on file systems and directory structures
  • YouTube tutorial: "Command Line Basics for Beginners"

Overall Experience

This assignment provided a hands-on way to learn command line navigation and file manipulation. I found it challenging but engaging, especially when:

Finding the hidden message in OldRecords.txt using the findstr command was like solving a real detective puzzle. The moment when I finally unlocked the door by copying the key file to the right location gave me a genuine sense of accomplishment.

The adventure structure made what could have been dry technical exercises into an immersive learning experience. I gained practical skills that will help me navigate file systems more efficiently in the future.

Creative Additions

To enhance my learning, I created my own set of quick reference flashcards for common command line operations. This helped me internalize the commands and their functions.

I also challenged myself by trying to complete the dungeon using only relative paths rather than absolute paths, which improved my understanding of directory relationships.

Additionally, I documented my adventure with a "commander's log" where I recorded each command used and its outcome. This helped me trace my steps and understand which approaches were most efficient.
