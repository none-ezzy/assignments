Available Commands

  • dir [options] - Displays a list of files and subdirectories
  • cd <dir> - Change directory (use cd .. to go up one level)
  • type <file> - Displays the contents of a text file
  • rename <old> <new> - Renames a file or files
  • tree - Graphically displays the directory structure
  • cls - Clears the screen
  • findstr - Searches for strings in files
  • attrib - Displays or changes file attributes
  • copy - Copies one or more files to another location
  • move - Moves files from one directory to another
  • del - Deletes one or more files
  • echo - Displays messages
  • title - Sets the window title
  • color - Sets console foreground and background colors
  • help - Shows help information

Challenge Tasks

  1. Start at the root directory (C:\) and explore the fortress
  2. Find and read the FortressLore.txt and README.md files
  3. Navigate through the EntranceGrounds to discover what lies beyond
  4. Find the key needed to unlock the LockedDoor
  5. Search for clues in the OldRecords.txt file
  6. Use the appropriate commands to unlock the door
  7. Reach the treasure hidden deep within the fortress

Hints

  • Use the tree command to get an overview of the dungeon structure
  • Read all text files carefully as they contain important clues
  • Pay special attention to Line 83 in the OldRecords.txt
  • Some doors may require special keys or commands to unlock
  • Remember: "All is not as it seems" - look for hidden meanings
